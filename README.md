# Для первого запуска нужно залить тему в магазин. Для этого: 
+ Упаковать файлы темы репозитория в архив с расширением .zip **(обязательно)**. 
+ Зайти в админку магазина и через "Theme Upload" указать архив и залить тему.

## Установите node-modules используя команду   в консоле:  npm install

## Что бы залить все файлы на сервер, используйте команду в консоле: npm run start

## Что бы включить отслеживание файлов, сборщик и залитие файлов в работе, которые вы изменяете, используйте команду в консоле: npm run watch
